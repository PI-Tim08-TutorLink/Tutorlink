# TutorLink

## Opis projekta
**TutorLink** je platforma koja spaja studente s tutorima i mentorima.  
Omogućuje rezervaciju termina, vođenje tutorijala online ili uživo, te evaluaciju i praćenje napretka studenata.

## Glavni korisnici
- Studenti  
- Tutori i mentori  
- Administratori platforme  

## Ključne funkcionalnosti
- Izrada korisničkog profila i vještina tutora  
- Pretraga i rezervacija termina s mentorima  
- Vođenje tutorijala putem video ili chat sustava  
- Praćenje napretka studenata  
- Sustav ocjenjivanja i povratnih informacija  
- Notifikacije o terminima  
- Administracija i upravljanje korisnicima
